Situação-problema inicial

Imagine que você precisa guardar as notas de 5 estudantes

Sem vetor, você poderia criar várias variáveis: nota1, nota2, nota3, nota4 e nota5.

Mas e se fossem 30 estudantes? E se fossem 100 produtos de uma loja? E se fossem as temperaturas de todos os dias do mês?

Criar uma variável para cada valor ficaria trabalhoso e confuso. Para resolver esse tipo de problema, usamos vetores.

Sem vetor

você precisa criar muitas variáveis, uma para cada novo valor.

nota1 ← 7.0
nota2 ← 8.5
nota3 ← 6.0
nota4 ← 9.0

Com vetor

você precisa criar apenas uma variável, que pode guardar de forma organizada muitos valores.

notas[0] ← 7.0
notas[1] ← 8.5
notas[2] ← 6.0
notas[3] ← 9.0
